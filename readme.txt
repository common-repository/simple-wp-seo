== Description ==

A simple way to add SEO tags to your posts and pages.

Add title and description manually, or let this plugin auto generate them for you.

Open Graph tags are automatically added for Facebook, LinkedIn etc.


== Installation ==

1. Install either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add tags on pages and posts, or let the plugin do the work for you!


== Frequently Asked Questions ==

= Do I have to add title and descriptions manually on each post/page for this plugin to work? =

No. Once the plugin is activated, title, description and other useful tags are added automatically to your pages and posts.

But if you want to, you can change the title and description individually on each post and pages.

= What are "Open Graph" tags for Facebook and LinkedIn? =

These tags are added to the page, so that Facebook and LinkedIn knows what to show when someone is sharing your page/post.

This adds title, description and even image to the posts.
